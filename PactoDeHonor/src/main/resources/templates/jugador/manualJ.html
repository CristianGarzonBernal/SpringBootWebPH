Manual del jugador

Requisitos mínimos

  • • Tarjeta gráfica Intel HD 4000
  • • Procesador Core i3 2.4 GHz
  • • 4 GB de RAM
  • • Windows 7, 8, o 10 de 64 bits

Requisitos recomendados

  • • Nvidia GTX 660 o AMD Radeon HD 7870 con GPU DX11 equivalente
  • • 2 GB VRAM
  • • Core i5 2.8 GHz
  • • 8 GB RAM
  • • Windows 7, 8, o 10 de 64 bits
